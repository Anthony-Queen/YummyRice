##  Features
-  **Wallpaper selector** as the source of truth
-  **Matugen theming across the entire system**
-   Consistent UI across all components
-   Smooth and responsive, tested on a T430S

##  Themed Components
- **Hyprland**
- **Waybar**
- **Wofi**
- **Dunst**
- **Kitty**

All dynamically recolored using Matugen.
